The main goal of this project was to increase the brand
awareness of the company and to create a user friendly,
easy to navigate through the webiste
With this project, which was another hackathon organized by Brainster Academy, we created the mobile version of the website for the fashion brand Zanov brand.


We were a team of five UX/UI designers, which we worked all the time together so we could create the website We started all together to do the research, to gather ideas, we also did a survey to get more answers from customers.
We started the project by researching the brand, its competitors and customers as well as its business goals.
We have created competitive analysis, survey, user flow and IA, user personas. Then we start to
I was involved in all the steps of creating the site, because here we worked as a team.
In the UI part the team and I chose the colors which we thought would best suit the design. We choosed the brown color because this color is a color that symbolizes connections, elegance, warmth, and honesty.
The whole project was another new experience for me, because it was not just a normal project but it was the biggest hackathon organized by Brainster on which I had the honor to work with colleagues from other academies.
During the hackathon I felt excitement, but also great fear that everything will not end on time. We created a mobile version of the website that we presented to the client.
